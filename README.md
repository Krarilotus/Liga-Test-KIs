# Liga AI Test Plugin

This repository contains the Liga AI test plugin for the Mod KI Team Liga / KI Turnier Liga setup for Stronghold Crusader.

## Purpose

The plugin bundles test AIs in UCP plugin format so they can be used and checked in the tournament environment.
