# Liga AI Test Plugin

This repository contains the test AI plugin for the Mod KI Team Liga / KI Turnier Liga setup for Stronghold Crusader.

## Purpose

The plugin bundles a small set of custom AIs in UCP plugin format so they can be tested and used in the Liga environment.

## Included AIs

- Al-Riyad
- Amira
- Bernard
- Bodo
- Buergermeister
- Charles
- Katatrina
- Ramsey
